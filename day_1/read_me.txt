cd name - перейти в папку
cd .. - перейти на уровень выше
mkdir name - создать папку
ls - показать содержимое текущей папки 
cp name_1 name_2 - создать копию файла name_1 с именем name_2
mv name_1 name_2 - переименовать/перезаписать файл name_1 в name_2
cat name - вывести содержимое файла
touch name - создать пустой файл name
ps - просмотр процессов Linux
man ps - мануал по команде ps
ps --help - справка по команде ps


nano name - создать/редактировать файл в nano
cnrtl+o - сохранить файл
cntrl+x - закрыть nano


git clone https:/... - клонировать репозиторий
git branch - посмотреть ветки
git branch name - создать ветку
git add name - добавить в commit файл name
git add . - добавить все файлы в commit
git commit -m "text" - добавить комментарий text в commit
git push origin master:develop - отправить commit на сервер
